# Janani Balamurali | Portfolio

A modern, responsive personal portfolio website showcasing my academic journey, technical skills, projects, certifications, and professional experience in Artificial Intelligence and Data Science.

## Overview

This portfolio serves as a central platform to present my work, technical expertise, and achievements. It is designed with a clean, responsive interface and highlights my interest in Artificial Intelligence, Data Analytics, Machine Learning, Full-Stack Development, and UI/UX Design.

## Features

- Modern and responsive design
- Dark and light theme support
- Interactive hero section
- Professional About section
- Education and Internship timeline
- Skills with proficiency indicators
- Featured projects
- Certifications and achievements
- GitHub profile integration
- Contact section with social links
- Smooth animations and transitions

## Technology Stack

- HTML5
- CSS3
- JavaScript (ES6)
- GitHub API
- Google Fonts
